ОБЯЗАТЕЛЬНОЕ ЗАДАНИЕ

Привести проект в соответствии с новым стандартом.
Переписать программу на классе.
Создать новый метод в классе, например eventsListeners. Перенести все действия, которые остались за классом внутрь него.
У нас есть методы, которые дублируют по факту один и тот же код(addExpensesBlock/addIncomeBlock и getAddIncome/getAddExpenses). Нужно создать 2 универсальных метода, которые будут принимать параметры в зависимости от которых будут происходить нужные нам действия.
Переменные, которые существуют только с неизменяемым параметром, сделать их объявление через const.


ДОПОЛНИТЕЛЬНОЕ ЗАДАНИЕ

Необходимо сохранять данные при перезагрузки страницы или после закрытия браузера. Для этого нам понадобиться cookie и localStorage
После нажатия кнопки Рассчитать то данные из правой части записываем в созданный вами localStorage и cookie(+ еще одну куку isLoad, которую нужно приравнять к true), для каждого значения отдельная куки.
При перезагрзки вставляются уже в правую часть. Левая часть так же остается заблокированной.
Если нажимаем сбросить, то из локального хранилища удаляется все
При выходе из браузер, перезагрузке все данные должны возвращаться в свои места(если они там были)
Если пользователь удаляет хотя бы одну из кук или она не соответствует тому, что хранинься в локал (кука name должна равнятся свойству name в локальном хранилище), тогда принудительно удаляем наши куки и локальное хранилище и программа запускается ПОЛНОСТЬЮ заново. (очищается объект от данных)
